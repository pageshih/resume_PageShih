Why I chose Front-End?

2019年開始，因為工作上需要使用的工具或技能不太需要變動，我開始焦慮於自己的成長停滯不前，另一方面也是因為看不到這份工作的前景，於是開始準備轉職。由於大學時的經歷，我對於數位相關的產業蠻有親切感，因為歷史系的專業也有田調的部分，所以參加了幾個 UX 的工作坊，也學習了跟排版設計較類似的 UI 課程，並參加了 HPX 的讀書會，學習 UI & UX 知識。

但是經過了 1 年的學習，在練習做 UI 時卻越來越困惑，因為學習越多知識就知道 UI 需要考量工程端能不能實現的問題，我不知道怎樣的稿子才是合理的，既然如此，那為何我不自己去寫看看呢？

2020年底，我報名了學米的 JavaScript 實戰營，雖然最終老師只是教切版而已，但我卻發現我寫程式的時候比做設計來得快樂許多，這成了我下定決心全力往前端轉職的契機，接下來我報名了六角學院的 JavaScript 直播班、切版班跟 Vue 班，不過因為一邊上班一邊上課，實在力不從心，學習也不太到位。2021 年 7 月，我正式離開我的出版社工作，當全職學習者，往前端工程的路邁進。
