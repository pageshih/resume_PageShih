Why I chose Front-End?

2019 年開始，我開始焦慮於自己的成長停滯不前，也因為看不到這份工作的前景，於是開始準備轉職。由於大學時的經歷，我對於數位相關的產業蠻有親切感，於是打算往與自己專業較為相近的 UI & UX 轉職，所以參加了幾個 UI & UX 的工作坊及線上課程，並加入 HPX 的讀書會進修。

但是經過了 1 年多的學習，在練習做 UI 時卻越來越困惑，因為學習越多，就知道 UI 需要考量工程端能不能實現的問題，我不知道怎樣的稿子才是合理的，於是我決定先在 hahow 上學習網頁知識，來解決這個問題。

2020 年底，我報名了學米的實戰營，學習怎麼切版，過程中我發現寫程式的時候比做設計來得快樂許多，這成了我下定決心全力往前端轉職的契機。接著我報名了六角學院的課程來加強我寫程式的能力，不過因為一邊上班一邊上課，實在力不從心，學習也不太到位。於是 2021 年 7 月，我正式離開我的出版社工作，規劃 1 年的時間來全心學習前端技術。
